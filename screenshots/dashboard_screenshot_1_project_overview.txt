[Description]
This screenshot shows the Project Overview page of the DentalVision-QA Streamlit 
dashboard. Key features include:

1. Top metric cards showing dataset statistics
2. Interactive bar chart of dataset splits  
3. Pipeline architecture visualization with status indicators
4. Clean healthcare-themed UI with professional color scheme
5. Responsive card-based layout with hover effects
6. Real-time data from the pipeline outputs

The dashboard is built with Streamlit and features:
- Interactive filtering and sorting
- Real-time chart updates
- Image preview functionality
- Export capabilities
- Mobile-responsive design
