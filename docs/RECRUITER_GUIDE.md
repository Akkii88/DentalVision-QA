### Technical

1. **CV Fundamentals:**
   - How does IoU work for object detection?
   - Why use YOLO over other detectors?
   - Explain non-maximum suppression

2. **ML/DL:**
   - How did you handle class imbalance?
   - What's your augmentation strategy?
   - How do you prevent overfitting?

3. **Quality:**
   - How does the QA/QC system work?
   - Why the 0.6 IoU threshold?
   - How do you handle ambiguous cases?
